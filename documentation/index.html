B) Installation - top

  • Unzip the downloaded folder
  • Copy all the files from the folder : "karakata" into your servers public_html or www directory. Just like in the image below
  • Update your correct database credentials in the .env file above
  • Open your browser and go to http://your-url/install
  • You would be taken to the install screen
  • Enter your admin settings and details including your purchase code.
  • Press the "Install" button.
  • That's it!

C) Setup - top

Once installation is complete. you are logged into the admin app. If not navigate to http://your-url/login and sign in with your admin credentials.

Locations

In the admin panel, go to the manage locations section and add locations from a country or individually

Categories

In the admin panel, go to the manage categories section. and add/remove categories to fit your use case

Settings

In the admin panel, go to the manage settings section. and update the app settings, such as site name and currency type to suit your use case

Payment

In the admin panel, go to the manage settings section. You can setup to begin receiving payment in the Paypal section by entering your paypal credentials
